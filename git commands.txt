git init - создание локального репозитория (проект)
git status - показывает изменения в репозитории (новые или редактированные файлы) 
git add . - добавит все изменения в репозиторий
git add file_name - добавит изменения конкретного файла
git commit -m "message" - добавит комментарий, к последним добавленным изменениям
git remote add origin <rep_url> - Подключает удаленный репозиторий (с github), в наш локальный репозиторий
git branch - показывает список сущетсвующих веток
git branch <branch_name> - переключает с одной ветки на другую
git push origin <branch_name> - отправляет локальные файлы, на удаленный сервер, на указанную ветку
git pull origin <branch_name> - подтягивает все обновления с удаленного репозитория
